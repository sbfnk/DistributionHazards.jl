ContactIntervals.ContactIntervals — Module

ContactIntervals.jl — parametric contact-interval distribution families.

A small shared kernel: the parametric distribution families for the contact interval as introduced by Kenah (2011) — the time from the onset of infectiousness in one individual to infectious contact with another. The hazard function h(τ) of a contact interval is the infectiousness kernel β(τ) at age-of-infection τ, the same object that drives the time-resolved within-household process in transtat and a contact-interval-parameterised branching-process simulation.

This package owns just the typed distribution families and their hazard / cumulative-hazard / survival / density (with log-scale variants) plus one derived helper, expected_cumhazard. Likelihoods, fitting, and simulation drivers belong to the downstream consumers.

References

  • Kenah, E. (2011). Contact intervals, survival analysis of epidemic data, and estimation of R₀. Biostatistics 12(3): 548–566.
  • Kenah, E. (2013). Nonparametric survival analysis of infectious disease data. Journal of the Royal Statistical Society, Series B 75(2): 277–303.

Types

ContactIntervals.ContactInterval — Type
ContactInterval

Abstract supertype for parametric contact-interval distributions in the sense of Kenah (2011): the time from the onset of infectiousness in one individual to infectious contact with another.

Subtypes are <: Distributions.ContinuousUnivariateDistribution. They need to define cumhazard, hazard, log_hazard, and Distributions.quantile. The rest — survival, density, log_density, and the pdf/cdf/ccdf/logpdf/logcdf/logccdf/rand interface from Distributions.jl — is derived once at this layer.

References

  • Kenah (2011), Biostatistics 12(3): 548–566.
ContactIntervals.ExponentialCI — Type
ExponentialCI(rate)

Exponential contact-interval distribution with rate λ = rate > 0.

h(τ) = λ, H(τ) = λ·τ, S(τ) = exp(-λ·τ).

The constant-hazard case; equivalent to WeibullCI(rate, 1) and LogLogisticCI(rate, 1) at shape 1 (each in their own family).

ContactIntervals.WeibullCI — Type
WeibullCI(rate, shape)

Weibull contact-interval distribution with rate λ = rate > 0 and shape γ = shape > 0.

H(τ) = (λτ)^γ, S(τ) = exp(-(λτ)^γ).

Reduces to ExponentialCI(rate) at shape == 1; the shape parameter controls whether infectiousness is increasing (γ > 1) or decreasing (γ < 1) over age-of-infection.

ContactIntervals.LogLogisticCI — Type
LogLogisticCI(rate, shape)

Log-logistic contact-interval distribution with rate λ = rate > 0 and shape γ = shape > 0.

S(τ) = 1 / (1 + (λτ)^γ), H(τ) = log(1 + (λτ)^γ).

A distribution with a non-monotone hazard (increases then decreases) for γ > 1 and a monotone-decreasing hazard for γ ≤ 1. Used in transreg for the external (community) component when the within-household process is exponential.

ContactIntervals.GammaCI — Type
GammaCI(rate, shape)

Gamma contact-interval distribution with rate λ = rate > 0 and shape γ = shape > 0.

Standard parametric form for Erlang-style infectious periods in compartmental epidemic models. Hazard is monotone increasing for γ > 1, monotone decreasing for γ < 1, and constant at γ = 1 (in which case GammaCI(λ, 1) == ExponentialCI(λ)).

H(τ) = -log Q(γ, λτ) and S(τ) = Q(γ, λτ), with Q the regularised upper incomplete gamma function. The cumulative hazard has no closed form; this implementation defers to Distributions.Gamma(shape, 1/rate) for logccdf and logpdf, so it inherits Distributions.jl's numerics and AD chain rules.

Hazard, cumulative hazard, survival, density

ContactIntervals.hazard — Function
hazard(d::ContactInterval, t::Real)

Hazard function h(τ) of the contact interval at age-of-infection τ = t. The infectiousness kernel β(τ) in the branching-process / renewal-equation view.

ContactIntervals.cumhazard — Function
cumhazard(d::ContactInterval, t::Real)

Cumulative hazard H(τ) = ∫₀^τ h(u) du of the contact interval.

E[H(ι)] over the infectious-period distribution gives the basic reproduction number R₀ in the branching (no-depletion) limit; see expected_cumhazard.

ContactIntervals.survival — Function
survival(d::ContactInterval, t::Real)

S(τ) = exp(-H(τ)).

ContactIntervals.density — Function
density(d::ContactInterval, t::Real)

f(τ) = h(τ) S(τ).

Log-scale variants

ContactIntervals.log_hazard — Function
log_hazard(d::ContactInterval, t::Real)

log h(τ). Defined per family for numerical stability; expected to be called only at strictly positive t (event times in survival analysis).

ContactIntervals.log_survival — Function
log_survival(d::ContactInterval, t::Real)

log S(τ) = -H(τ); evaluated as one expression for AD/numerical stability.

ContactIntervals.log_density — Function
log_density(d::ContactInterval, t::Real)

log f(τ) = log h(τ) + log S(τ). Requires t > 0.

Derived helpers

ContactIntervals.expected_cumhazard — Function
expected_cumhazard(ci::ContactInterval, infectious_period::Distribution;
                   atol=1e-10, rtol=1e-8) -> Float64

Expectation E[H(ι)] of the contact-interval cumulative hazard H under the infectious-period distribution ι ~ infectious_period, computed by adaptive Gauss-Kronrod quadrature.

In the branching (no-depletion) limit this is the basic reproduction number R₀; see Kenah (2011), eq. 3.1. It is also the normalising constant of the contact-age density driving the generation interval.

atol and rtol are forwarded to QuadGK.quadgk.
